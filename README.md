The Intelligent Alzheimer’s Risk Prediction Model is a machine learning–based system designed to predict the risk percentage of Alzheimer’s disease. This model helps in the early detection and diagnosis of the disease by analyzing medical data and brain images.The system uses machine learning algorithms such as the Multinomial Naive Bayes classifier along with the MobileNet deep learning model to analyze patterns related to Alzheimer’s disease. MobileNet is used for efficient feature extraction from MRI brain images, while the Multinomial Naive Bayes algorithm is used for classification and prediction of Alzheimer’s risk.
